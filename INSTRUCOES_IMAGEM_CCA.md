# Instruções para adicionar a imagem do Informativo CCA

1. Salve a imagem do informativo CCA que foi colada na conversa
2. Renomeie o arquivo para: `cca-informativo.jpg`
3. Faça upload do arquivo para: `/workspaces/SSMA_TUB/assets/images/cca-informativo.jpg`
4. Delete este arquivo após concluir

A imagem deve ser colocada no diretório: `/workspaces/SSMA_TUB/assets/images/`

O card de CCA já está configurado com:
- Botão WhatsApp: 27988175824
- Botão Ver Informativo: abre a imagem em modal
